Скринсейвер с трёхмерной вращающейся надписью, которая движется по экрану и отскакивает от границ

Описание:

При запуске пользователь вводит текст, который отображается как 3D-надпись. Она вращается вокруг вертикальной оси и движется по экрану, отскакивая от границ. Размер букв подстраивается автоматически

Классы:

Main - точка входа. Принимает текст от пользователя, создаёт окно и запускает рендер

Renderer - управляет анимацией и отрисовкой. Каждые 16 мс вращает вершины, двигает объект, проверяет границы и перерисовывает сцену

Letter - хранит геометрию буквы (вершины и грани)

Vector - точка в трёхмерном пространстве. Хранит координаты x, y, z и реализует вращение вокруг оси Y

Face - грань объекта. Хранит индексы вершин, цвет и глубину для сортировки (использует алгоритм художника)

Ограничения:

Реализованы буквы: А, Г, Н (+ латинские A, H)

Остальные символы пропускаются, оставляя пустое место

Рекомендуемая длина слова до 10 букв (слова больше 10 букв обрезаются)

Запуск:

Требования: Java 8 и выше

javac ru/kentolf/screensaver/*.java

java ru.kentolf.screensaver.Main

После запуска появится окно для ввода слова. Если ввод пустой или отменён, то отображается надпись AAA
